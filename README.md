# Pusula

Profil ve ilgi alanlarına göre teknoloji/kariyer rotası önerileri sunan web uygulaması.

## Canlı demo

**https://pusula-app-two.vercel.app**

(Başvuru veya jüri için bu adresi kullanın; Vercel production domain’idir.)

## Geliştirme

Ön yüz `web/` klasöründedir.

```bash
cd web
npm install
npm run dev
```

Ortam değişkenleri için `web/.env.example` dosyasına bakın.
